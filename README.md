Writing tests suits for a feed reader project:

There is 7 specs that are:

Initial Entries
  * there is an element in the feed container
New Feed Selection
  * new feed changes
RSS Feeds
  * name defined
  * url defined
  * are defined
The menu
  * menu element is hidden
  * change visibility when clicked
